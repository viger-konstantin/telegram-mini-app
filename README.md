# telegram-mini-app

Мини-приложение для Telegram — стартовая структура проекта для разработки мини-приложения/бота.

## Что внутри

- README.md — эта инструкция
- package.json — базовая конфигурация npm
- src/index.js — заглушка для точки входа
- .gitignore — типичные исключения
- LICENSE — лицензия MIT

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `npm install`
3. Запуск: `npm start`

## Структура

src/
  └─ index.js  — точка входа
